
Red Line Project
================

A Symfony project started on April 1(:-), 2016.

## Installation:

Require Symfony2 and Composer.
```
$ Git clone http://github.com/celkae/fil_rouge.git
```
```
$ cd fil_rouge
```
```
$ composer Install
```
```
$ app/console server:run
```
